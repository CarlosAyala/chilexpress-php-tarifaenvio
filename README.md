# chilexpress-php-tarifaenvio
<p>Clase PHP para calcular tarifa de envío de encomiendas para servicio de Chilexpress</p>

# Requerimientos
<p>PHP5.6+</p>
<p>Extension SOAP para php</p>

# Información
